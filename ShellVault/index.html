How to Add Your Script?

Want to contribute to ShellVault?

  1. Clone the repository
  2. Edit data/scripts.json to add your script
  3. Place your .sh file in files/ folder
  4. Create a Pull Request!

Remember to add SHA256! Generate it with: sha256sum <file.sh>
